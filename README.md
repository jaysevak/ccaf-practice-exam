# CCAF Practice Exam Simulator

A free, open-source practice exam for the **Claude Certified Architect — Foundations** certification.

🔗 **Live:** [https://jaysevak.github.io/ccaf-practice-exam/](https://jaysevak.github.io/ccaf-practice-exam/)

## Features

- **Timed Exam Mode** — 60 questions, 120-minute timer, 1000-point scoring (720 to pass)
- **Practice Mode** — Untimed, instant explanations after each answer
- **Domain Drill** — Focus on your weak areas
- **Questions & options shuffle every session** — no memorizing positions
- **Full review** with correct answers + detailed explanations
- **Keyboard shortcuts** — 1-4 pick answer, ←/→ navigate, F flag
- **Works offline** — single HTML file, no backend, no login

## Exam Domains

| Domain | Weight |
|--------|--------|
| Agentic Architecture & Orchestration | 27% |
| Tool Design & MCP Integration | 18% |
| Claude Code Configuration & Workflows | 20% |
| Prompt Engineering & Structured Output | 20% |
| Context Management & Reliability | 15% |

## How to Use

**Online:** Visit [https://jaysevak.github.io/ccaf-practice-exam/](https://jaysevak.github.io/ccaf-practice-exam/)

**Offline:** Download `index.html` and open it in any browser.
